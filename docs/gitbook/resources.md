## Glossary

### Common Terms
- **FHE**: Fully Homomorphic Encryption
- **euint**: Encrypted unsigned integer
- **Handle**: Reference to encrypted value
- **Input Proof**: Zero-knowledge proof of encryption
- **Gas**: Cost to execute transactions
- **Testnet**: Test blockchain network
- **Smart Contract**: Self-executing code on blockchain
- **Solidity**: Smart contract programming language

### FHEVM Specific
- **fhevm**: Zama's FHE virtual machine
- **FHE.allow()**: Grant decryption access
- **FHE.allowThis()**: Allow contract to use encrypted value
- **fhevmInstance**: Client-side FHE instance
- **Zama Config**: Network configuration
- **Hardhat Plugin**: Testing integration
